LESSON OVERVIEW LESSON 1 -- DIFFERENTIATION LESSON 2 -- INTEGRATION LESSON 3 -- SUMMATION

2026 will be the second year that students taking the AP Calculus exams (AB & BC) will have access to the Desmos graphing calculator. Complete these lessons to learn the must-know functions to maximize your success on the exam. The first two lessons assume that you already have first semester (Calc 1 or AB) skills in hand, but the final lesson is from second semester (Calc 2 or BC) skills, so don't worry if summation is new to you.

For each lesson, you will use the embedded Desmos calculator to follow the instructions. At the end of the lesson is a self-checking quiz question. Enter your answers into the quiz to verify that you performed the skills correctly. Important: Note that the calculators embedded in this site do not retain their state, so your work will not be viewable by anyone, yourself included if you navigate away from the page. As a result, be sure to check your work before moving on to the next lesson.

As is the expectation on either of the AP Calculus exams, students should use three-digit accuracy. To practice this skill before beginning the first actual lesson, follow these steps using the calculator.

  1. On line 1, define a cubic function by typing "f(x)=x^3".
  2. On line 2, define "a=sqrt{17}".
  3. On line 3, input the rounded previous value into the function by typing "f(4.123)".
  4. On line 4, input the entire previous value into the function by typing "f(a)".
  5. In the textbox below the calculator, input this value rounded to three decimal place accuracy. (Note that there is indeed a difference.)
  6. Click the button to confirm that you found this value correctly.
